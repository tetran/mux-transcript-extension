# Laravel Learn Mux Subtitle

Laravel Learn のレッスンページ（`https://laravel.com/learn/*`）で、Mux Player の再生時間に同期して Transcript タブの字幕を動画直下にオーバーレイ表示する Chrome 拡張機能です。

## 機能

- 動画の再生時間に合わせて字幕を自動表示
- DeepL Free API による自動日本語翻訳（オプション）
- ブラウザ標準のページ翻訳にも対応

---

## 開発者インストール

Chrome の拡張機能として直接読み込む方法です。Chrome Web Store への公開は不要です。

### 手順

1. このリポジトリを ZIP でダウンロードして解凍する（または `git clone` する）
2. Chrome のアドレスバーに `chrome://extensions/` を入力して開く
3. 右上の「デベロッパーモード」を ON にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. 解凍したフォルダを選択する

読み込みが完了すると、拡張機能一覧に「Laravel Learn Mux Subtitle」が表示されます。

### アップデート方法

ファイルを更新した後、`chrome://extensions/` の拡張機能カードにある更新ボタン（丸矢印アイコン）をクリックするとリロードされます。

## つかいかた

### A. DeepL Free API を使った自動翻訳（推奨）

DeepL の無料枠（月 50 万文字）を使って、Transcript テキストを自動で日本語に翻訳します。

1. DeepL API の無料登録

   以下のページから「無料で登録する」を選択してアカウントを作成します。
   https://www.deepl.com/ja/pro#developer

2. API キーを発行・取得

   登録後、DeepL のアカウントページから API 認証キーを取得します。

3. 拡張機能のオプション画面で API キーを設定する

   設定方法は後述の「[オプション画面の使い方](#オプション画面の使い方)」を参照してください。

4. Laravel Learn のレッスンページを開く

   例: https://laravel.com/learn/getting-started-with-laravel/what-are-we-building

   ページがロードされると自動的に翻訳・字幕表示が行われます。

5. 動画を再生する

   再生位置に合わせて字幕が切り替わります。

### B. ブラウザ標準のページ翻訳を使う

DeepL API キーなしでも、Chrome のページ翻訳機能と組み合わせて使えます。

1. Laravel Learn のレッスンページを開く

   例: https://laravel.com/learn/getting-started-with-laravel/what-are-we-building

2. ページ内で右クリックし、「日本語に翻訳」を選択する
3. Transcript が日本語に翻訳されたことを確認する
4. 動画を再生する

拡張機能は翻訳後の表示テキストをそのまま拾って字幕表示します。

## オプション画面の使い方

拡張機能のオプション画面では、翻訳の挙動や字幕表示を細かく設定できます。

### オプション画面を開く方法

`chrome://extensions/` で「Laravel Learn Mux Subtitle」の「詳細」→「拡張機能のオプション」をクリックします。

### 設定項目

#### DeepL API キー

DeepL Free API の認証キーを入力します。設定するとページロード時に自動翻訳が有効になります。空のままにすると、ブラウザ翻訳モード（方法 B）で動作します。

#### 翻訳のていねいさ

日本語訳の文体を3段階から選択できます。

| 選択肢 | 説明 |
|--------|------|
| ていねい | 丁寧語・敬語を使った訳文 |
| ふつう | 標準的な話し言葉 |
| カジュアル | くだけた口語表現 |

#### 話し方（任意）

訳文のトーンや雰囲気を自由にテキストで指定できます（最大 300 文字）。
例: `コメディアンっぽく`、`法廷の雰囲気で`、`ゆるくフレンドリーに`

空欄のままにすると、「ていねいさ」の設定のみが適用されます。

#### 字幕最小表示時間

字幕が短時間で頻繁に切り替わるのを抑えるための設定です（デフォルト: 2秒）。
数値を大きくすると字幕の切り替えが緩やかになります。
